ปรับแก้ไขแอพตามเงื่อนไขนี้ เพิ่มเติมครับ 
1. หน้าสมัครเรียน ให้สามารถ login auto ผ่าน line id ของสมาชิก โดยไม่ต้องใส่รหัสผ่านเอง (ตามภาพที่ส่งไป)
2. หน้านี้ให้เพิ่มอีกส่วน upload slip bank ที่ส่งภาพให้ (สามารถแก้ไขเปลี่ยนได้เอง โดย admin หลังบ้าน พร้อมใส่คำแนะนำไทย)
3. Logo The Owner ตามภาพที่ส่งไปให้

โดยที่ แอพสามารถใช้งานได้ทุก platform รวมถึง laptop , mobile phone และสามารถปรับแต่งได้ ผ่าน admin page เช่น color theme,font size
โดยที่เลือกใช้ระบบที่ฟรีเป็นหลัก เพราะจำนวนคนประมาณ xxxx คน 
ระบบที่อยากให้ใช้ google sheet, google form , app script , line OA , firebase (ขอระบบฟรีเป็นหลักก่อน และตัวระบบสามารถปรับเปลี่ยนได้ ถ้าคนเพิ่มมากขึ้น และไม่ช้า!)

การทำงานโดยรวม จะมี input 2 แบบ
    1.หน้า landing page เพื่อรับข้อมูล และส่งต่อไปที่ google sheet , line oa "The Owner"
    2.Line OA "The Owner" จะเพิ่ม rich menu เพื่อทำระบบนี้เข้าไป (จะทำเพิ่มหลังจากระบบทุกอย่างพร้อมรัน production)

แบบสำหรับให้สมาชิกสมัครจะมี 2 แบบ เรียนได้ทั้ง online , onsite
    1.แบบรายครั้ง (trial) = 150 บาทต่อครั้ง (จะเริ่มนับเวลา หลังจาก checkin เข้าเรียนครั้งแรก เรียนได้เพียง 1 คอร์ส หรือ 1 วัน)
    2.แบบราย 3 เดือน (quarter) = 600 บาทต่อครั้ง (จะเริ่มนับ หลังจาก checkin เข้าเรียนครั้งแรก + 90 วัน)

- ทุกครั้งที่สมาชิกสมัคร ,ลงทะเบียนเข้ามาให้ สร้างรหัสประจำทีม The Owner คือ Own+(T,P)+xxxx = OwnT0001 , OwnP0001
    - โดยที่รหัสประจำทีมจะสร้างเรียงตามเลขลำดับไปเรื่อยๆ จาก 0 ไปถึง 9999 
    - T = Trial (สมาชิกที่สมัครแบบรายครั้ง)
    - P = Package quarter (สมาชิกที่สมัครแบบราย 3 เดือน)
    - xxxx = ตัวเลขรันไปเรื่อยๆ จาก 0-9999 (โดยที่จะทำการลบรหัสประจำตัว "member ID The Owner" ออกหลังจากที่สมาชิก หมดอายุ การเป็นสมาชิกแล้วเป็นเวลา 3 เดือน  โดยจะมีแจ้งหลังบ้านให้ admin ทราบ เพื่อให้ admin ทำการกดปุ่มลบโดยง่าย )
    - รหัสประจำทีม จะอยู่ใน sheet หลังบ้านที่ admin สามารถเข้าไปดู,บริหารจัดการได้ง่าย

มี link หน้า check-in เพื่อให้สมาชิกเข้าไปลงชื่อเรียนแต่ละคอร์สล่วงหน้าได้
ถ้ามาเรียน onsite สามารถเช็คอินล่วงหน้าเพื่อจองที่นั่งแบบ onsite ถ้ามีจำกัด seats แต่ถ้ามาแบบฉุกเฉินผ่านอาจจะ check inผ่าน qr code หน้าห้องได้ ผ่าน admin แต่อาจจะต้องมี ค่าปรับ xxx บาท เพื่อไม่ให้มาฉุกเฉินแบบนี้อีก

ตอนนี้ ต้องการให้ สมาชิกเข้ามาผ่าน landing page หรือ line oa "the owner" แล้วส่ง slip bank เพื่อเช็คข้อมูลโดย admin ก่อน ที่จะส่งข้อมูลต่างๆ กลับไปที่ สมาชิก ผ่าน line oa และมีเก็บข้อมูลเพื่อตรวจสอบผ่าน google sheet เพื่อให้ admin ตรวจสอบ แก้ไขหลังบ้านได้ 

ถ้าเรียนแบบ online จะเรียนผ่าน zoom application เป็นหลัก จะต้องมีการเช็คสิทธิ์ สมาชิกว่าสมัครแบบไหน สมัครหรือยัง ผ่าน "member ID The Owner" ของมือถือสมาชิกแต่ละคน จะมีการเช็คสิทธิ์ กันปัญหาแอบเข้ามาเรียน โดยไม่ได้สมัครสมาชิก แบบง่ายๆ admin สามารถ บริหารหลังบ้าน ผ่าน google sheet หรือ landing page,admin page ได้
