FAQ คำถามที่พบบ่อย
มีข้อสงสัยเพิ่มเติม ติดต่อได้ผ่าน LINE OA
เริ่มนับอายุสมาชิกเมื่อไหร่?▼
อายุสมาชิกจะเริ่มนับตั้งแต่วันที่ Check-in เข้าเรียนครั้งแรก — Trial นับ 1 วัน, Quarter นับ 90 วัน
เรียน Online ผ่านช่องทางไหน?▼
เรียนผ่าน Zoom Application — หลังจาก Admin Approve จะส่ง Meeting ID และ Password ให้ผ่าน LINE OA โดยอัตโนมัติ
ถ้าจองแล้วไม่มา มีค่าปรับไหม?▼
มีค่าปรับสำหรับการไม่มาโดยไม่แจ้งล่วงหน้า: Online ฿20 | Onsite ฿50 | Hybrid ฿100 กรุณายกเลิกล่วงหน้าอย่างน้อย 2 ชั่วโมง
สมัครผ่าน LINE ได้ไหม?▼
ได้ครับ! กดปุ่ม "เข้าผ่าน LINE" ระบบจะดึงข้อมูลชื่อและ LINE ID อัตโนมัติ ไม่ต้องจำรหัสผ่าน
รหัสสมาชิก OwnT/OwnP คืออะไร?▼
รหัสประจำตัวที่ระบบสร้างให้อัตโนมัติ: OwnT = Trial, OwnP = Quarter ตามด้วยเลข 0001-9999 ใช้สำหรับ Check-in และยืนยันสิทธิ์การเรียน
Admin Approve นานแค่ไหน?▼
ปกติภายใน 24 ชั่วโมง (วันทำการ) หลังส่ง Slip ถ้าเร่งด่วนติดต่อได้โดยตรงผ่าน LINE OA @theowner
